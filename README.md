# Dinámica de la Atmósfera

Bienvenido a este espacio dedicado al estudio de la dinámica que gobierna la atmósfera terrestre.  
La atmósfera no es solo un fluido en movimiento: es un sistema dinámico complejo donde convergen la rotación planetaria, la estratificación, la convección, las ondas y procesos no lineales que determinan el tiempo y el clima.

Este repositorio reúne los cuadernos de Jupyter utilizados en el curso de **Dinámica de la Atmósfera**, donde combinamos teoría matemática rigurosa con un poco de experimentación computacional.

Aquí no solo resolvemos ecuaciones: buscamos entender los principios físicos que organizan la circulación atmosférica.

---

## Cómo usar este repositorio

Los notebooks pueden ejecutarse de manera local si tienes Python instalado o directamente en la nube usando MyBinder, sin necesidad de instalar ningún software adicional.

---

## Ejecutar en la nube con MyBinder

### Acceso directo

Haz clic en el siguiente botón para abrir el repositorio en Binder:

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/ajaraminimbus/dinamica-atmosferica-icaycc/main)

Binder construirá automáticamente el entorno utilizando el archivo `requirements.txt` del repositorio. El primer inicio puede tardar algunos minutos mientras se genera el entorno.

---

### Acceso manual paso a paso

1. Ir a https://mybinder.org  
2. En GitHub repository name or URL escribir:

   ajaraminimbus/dinamica-atmosferica-icaycc

3. En Git ref (branch, tag, or commit) escribir:

   main

4. Hacer clic en Launch.

---

## Notas importantes

El entorno se genera dinámicamente a partir del repositorio en GitHub.  
Cada sesión es temporal, por lo que los cambios realizados no se guardan permanentemente.  
Si la sesión permanece inactiva durante un tiempo prolongado, se cerrará automáticamente.
